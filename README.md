## Overview

This project predicts crop yield categories using Machine Learning techniques and SQL-based data analysis.

The workflow includes:

- Data Cleaning
- Feature Engineering
- SQL Analytics
- Exploratory Data Analysis
- Machine Learning
- Model Evaluation

---

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- SQL
- Jupyter Notebook

---

## Dataset

- Records: 1,378
- Features: 16
- Target:
  - Low Yield
  - Medium Yield
  - High Yield

---

## Models

- KNN Classifier
- Random Forest Classifier

Best Model:
Random Forest

---

## Workflow

1. Data Cleaning
2. SQL Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Prediction
